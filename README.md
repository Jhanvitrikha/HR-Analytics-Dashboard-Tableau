📊 HR Analytics Dashboard – Tableau Project
📌 Project Overview

This project presents an Interactive HR Analytics Dashboard developed using Tableau.
The dashboard provides meaningful insights into employee data such as attrition rate, salary distribution, department-wise analysis, and workforce demographics.

The objective of this project is to support HR teams and management in making data-driven decisions by transforming raw employee data into clear and actionable visual insights.

🛠 Tools & Technologies Used

Tableau Desktop

HR Dataset (.hyper file)

Data Visualization Techniques

Business Intelligence Concepts

📂 Project Files

Tableau_HR_Dashboard.twb – Tableau Workbook File

HR Data (HR Data).hyper – Data Source File

📈 Dashboard Features

✔ Overall Employee Count & KPIs
✔ Attrition Analysis
✔ Department-wise Employee Distribution
✔ Gender Diversity Analysis
✔ Salary Distribution Insights
✔ Interactive Filters for Dynamic Exploration

📊 Key Insights

Identified attrition trends across different departments

Analyzed salary patterns among employees

Observed workforce gender distribution

Provided department-level workforce breakdown

🚀 How to Use

Download the repository files

Open Tableau_HR_Dashboard.twb in Tableau Desktop

Connect to the .hyper data file if prompted

Explore the interactive dashboard
